# SUPLEFEMME — Landing

Landing institucional do SUPLEFEMME (Next.js 14 + TypeScript + Tailwind).

## Como rodar

```bash
pnpm install   # ou: npm install / yarn
pnpm dev       # http://localhost:3000
```

## TODOs do cliente (preencher antes de publicar)

- `WHATSAPP_NUMBER` em `lib/config.ts`
- CNPJ, endereço e contato no `components/sections/Footer.tsx`
- Logo em SVG (hoje é tipografia)
- Render oficial do pote (substituir PNGs em `public/product/` se desejar)
- URLs reais de Política de Privacidade e Termos de Uso
- Eventual nº de notificação ANVISA
